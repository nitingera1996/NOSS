Introduction

Welcome to the NOSS API! You can use our API to access NOSS API endpoints, which can get information on Rail Routes, Air Routes available you to from your current location to your final destination entered by you. You can also plan your journey with the number of days available to you; go through our predictions and come back within the time limites to various cities available according to your requirements.

Authentication

NOSS uses a all together method of authentication using public key and private key. First user need to register here. He will get a public key which he need that to store secretly(just like api key) will sending request he need to encrypt the orgin address with that key and pass that encrypted string under parsed text.

Link to the algorithm is here.

APIs

Rail Route API

Purpose: Get Rail Route

Pattern: http://localhost:8000/railroute/?departure_date=<departure date>&origin=<origin address>&duration=<duration>&public_key=<public key>y&parsed_string=<parsed string>

Parameter
Description
railrouteApp name
departure_dateDeparture Date (Format: YYYY-MM-DD)
originOrigin Address (May Be Your Home Address)
durationTime Duration
public_keyPublic Key
parsed_stringParsed String

Description
airrouteApp name
